# Student Performance Prediction System

## Introduction

This project uses Machine Learning to predict a student's final performance based on attendance, marks, and study hours.

## Objective

To build a predictive model that estimates student performance using academic factors.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Matplotlib

## Dataset

The dataset contains:

* Attendance
* Marks
* Study Hours
* Final Score

## Methodology

1. Data Collection
2. Data Preprocessing
3. Train-Test Split
4. Linear Regression Model Training
5. Prediction
6. Accuracy Evaluation
7. Data Visualization

## Results

The model predicts student performance and displays accuracy using the R² score.

## Conclusion

This project demonstrates how Machine Learning can be used to analyze and predict student performance.
