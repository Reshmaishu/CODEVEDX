# AI Based Fake News Detection Tool

## Overview
This project is an NLP-based Fake News Detection System developed using Python and Machine Learning. The application analyzes news text and predicts whether it is Fake or Real while providing a confidence score.

## Features
- News text input
- Text preprocessing
- TF-IDF vectorization
- Fake/Real classification
- Confidence score output
- Trained model storage using Pickle

## Technologies Used
- Python
- Pandas
- NLTK
- Scikit-learn
- Matplotlib
- Pickle

## Project Workflow
1. Data Collection
2. Text Preprocessing
3. TF-IDF Vectorization
4. Model Training
5. Model Evaluation
6. Model Storage
7. News Prediction

pip install -r requirements.txt

python main.py
